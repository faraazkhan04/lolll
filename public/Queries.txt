Delete command
The DELETE command removes tuples from a relation. It includes a WHERE clause, similar to that used in an SQL query, to select the tuples to be deleted.
>>>DELETE FROM EMPLOYEE WHERE LNAME='Brown';
>>>DELETE FROM EMPLOYEE

Update Command
The UPDATE command is used to modify attribute values of one or more selected tuples. An additional SET clause in the UPDATE command specifies the attributes to be modified and their new values.
An example is to give all employees in the 'Research' department a 10 percent raise in salary
